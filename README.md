# solarwinds-api2

# Description

PowerShell Module that interfaces with the solarwinds orion api2 perfstack endpoint (/api2/perfstack)

## Getting Started

Download the SolarWinds-PerfStackAPI.psm1
Import-Module SolarWinds-PerfStackAPI.psm1
